TotalPoint V.1 (WPF)
TotalPoint คือโปรแกรม Desktop Application ที่พัฒนาด้วย C# WPF (.NET 10) ออกแบบมาเพื่อช่วยช่างรังวัดและช่างโยธาในการจัดการข้อมูลหมุดหลักเขตที่ดิน โดยเน้นการแปลงข้อมูลดิบให้เป็นรูปแบบที่พร้อมใช้งานกับกล้องสำรวจ (Total Station) และโปรแกรม DOLCAD

คุณสมบัติหลัก
Import Data: รองรับการนำเข้าไฟล์ .txt และดึงข้อมูลจุดตั้งกล้อง (Station), จุดธงหลัง (Backsight) และระยะทางโดยอัตโนมัติ

Smart Name Management: จัดการรายชื่อหมุดหลักเขตได้ง่ายขึ้น โดยสามารถดับเบิลคลิกที่ชื่อหมุดเพื่อแก้ไขได้ทันที และมีระบบป้องกันชื่อหมุดซ้ำ

Calculation Engine:

แปลงค่ามุมและระยะทางเป็นรูปแบบ Record สำหรับกล้องสำรวจ

ระบบจำลองค่าความคลาดเคลื่อน (Error Simulation) เพื่อให้ข้อมูลมีความเป็นธรรมชาติ

รองรับการเข้ารหัสชื่อภาษาไทย (Thai Name Encoding) เพื่อป้องกันปัญหาการแสดงผลบนอุปกรณ์ที่ไม่รองรับภาษาไทย

Multi-Format Export:

ส่งออกไฟล์ .scr สำหรับรัน Script ใน DOLCAD

ส่งออกไฟล์ .txt สำหรับเก็บสำรองข้อมูลหมุด

Visual Output: แสดงผลลัพธ์การคำนวณผ่าน RichTextBox พร้อมระบบแยกสี Keyword (STA, BS, SS) เพื่อให้อ่านข้อมูลได้ง่าย

โครงสร้างโปรเจกต์
Wpftotal1.slnx: ไฟล์ Solution ของโปรเจกต์

MainWindow.xaml / .cs: ส่วนติดต่อผู้ใช้และตรรกะการทำงานหลักของโปรแกรม

App.xaml: การตั้งค่าเริ่มต้นของแอปพลิเคชัน

Wpftotal1.csproj: ไฟล์กำหนดคุณสมบัติของโปรเจกต์ (SDK: Microsoft.NET.Sdk, Target: net10.0-windows)

ความต้องการของระบบ
.NET 10.0 Runtime (Windows)

ระบบปฏิบัติการ Windows (เนื่องจากเป็น WPF Application)

การติดตั้งและใช้งาน
